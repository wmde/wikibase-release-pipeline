Wikibase Cloud Init Generator

This tool helps you generate a cloud-init.yaml file to deploy your own Wikibase instance on a VPS (e.g. Hetzner, DigitalOcean, AWS). Just paste the generated file into your cloud provider's User Data field when creating a server.

All fields are optional — if you leave any blank, the setup script will generate secure defaults automatically, including passwords and domain names based on your server’s IP address (using traefik.me).

Features

  • 🔧 Fully automated Wikibase installation
  • 🌐 Optional default domains like IP.traefik.me and query.IP.traefik.me
  • 🔐 Secure passwords generated automatically if left blank
  • 🐳 Powered by Docker + Docker Compose
  • 🚀 One-click bootstrap via cloud-init
